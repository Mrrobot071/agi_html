# E o Pix?

Este é um projeto simples que apresenta uma página web com uma pergunta engraçada e interativa: "Vai me pagar não?". Os usuários podem escolher entre "Sim" e "Não", e dependendo da escolha, uma ação é desencadeada.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Exibe uma pergunta "Vai me pagar não?".
- Permite que os usuários escolham entre "Sim" e "Não".
- Responde de acordo com a escolha do usuário.
- Redireciona para um link externo se a escolha for "Sim".
- Ao clicar no botão "Não", ele muda de posição aleatoriamente. 
## Pré-requisitos

Não são necessários pré-requisitos especiais para rodar este projeto. Basta abrir o arquivo `index.html` em um navegador web.

## Como Usar

1. Clone este repositório ou faça o download dos arquivos.
2. Abra o arquivo `index.html` em seu navegador.
3. Responda à pergunta clicando em "Sim" ou "Não".
4. Divirta-se!
